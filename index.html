Percentage Image Maper

Instructions

  1. Select an image with the file picker below
  2. Double-click anywhere on the image to create a map area
  3. Click on the created map area to select it
  4. Use the square handles to resize or move the map area
  5. Multiple map areas can be added (colors will repeat after 6)
  6. Tables will dynamically generate below for each map area
  7. You can right-click the canvas to save the image with mapped areas

Limitations

  • Use Google Chrome for best performace
    (Firefox and Safari should work, IE may not)
  • Only 1 map area can be selected, moved, or resized at a time
If this text is displayed, your browser does not support HTML5 Canvas.
